# 交互课件 Demo 项目说明

访问地址: https://chen-demo.vercel.app/

本项目包含六个 HTML 文件，演示了不同类型的交互式课件页面。以下是每个文件的详细说明：

## 1. 首页导航 (index.html)

- **内容说明**: 项目的入口页面，提供导航菜单以跳转到各个演示页面。
- **使用图片**:
  - `assets/select/bj.jpeg`: 背景图片。
- **功能**:
  - 展示四个功能按钮（选词填空、连线题、消除题、选择题）。
  - 按钮具有入场动画和鼠标悬停特效。
  - 点击按钮跳转至对应的 HTML 页面。

## 2. 选择题 (select.html)

- **内容说明**: 一个标准的单项选择题页面，题目为“面对飞岛的袭击，地面的人们的态度是怎样的？”。
- **使用图片**:
  - `assets/select/bj.jpeg`: 背景图片。
  - `assets/select/bt.png`: 标题区域背景图。
  - `assets/submit.png`: 提交按钮图标。
- **功能**:
  - 用户点击选项（A、B、C）进行单选，选中项会有高亮样式。
  - 点击提交按钮后，系统判断对错并弹出相应的反馈提示（🎉 回答正确 或 ❌ 遗憾）。

## 3. 连线题 (connect.html)

- **内容说明**: 左右两列连线配对题（例如：匹配相同的颜色词汇）。
- **使用图片**:
  - `assets/connect/bj.jpeg`: 背景图片。
  - `assets/connect/title.png`: 标题区域背景图。
  - `assets/submit.png`: 提交按钮图标。
  - `assets/reset.png`: 重置按钮图标。
- **功能**:
  - 用户点击左侧和右侧的选项进行配对。
  - **SVG 连线**: 选中两个选项后，页面会使用 SVG 动态绘制一条连接线。
  - **逻辑控制**: 支持更改连线；点击提交按钮检查连线是否正确；点击重置按钮清空所有连线。

## 4. 消除题 (eliminate.html)

- **内容说明**: 一个趣味消除小游戏，画面中有三个随机移动的人物角色。
- **使用图片**:
  - `assets/eliminate/bj.png`: 背景图片。
  - `assets/eliminate/a.png`, `b.png`, `c.png`: 三个不同的人物/角色图片。
  - `assets/reset.png`: 重置按钮图标。
- **功能**:
  - **随机移动**: 人物图片每 5 秒在屏幕特定区域内随机变换位置，且有平滑过渡动画。
  - **消除交互**: 点击人物图片，该人物即被消除（隐藏）。
  - **胜利判定**: 当所有人物都被消除后，弹出“全部消除”的提示。
  - 支持重置游戏。

## 5. 选词填空 (fill.html)

- **内容说明**: 一个图文结合的填空题。句子为：“走进一瞧，这里有 [ ] ，正在游来游去地嬉戏。”
- **使用图片**:
  - `assets/fill/bj.png`: 背景图片。
  - `assets/fill/LaoShu.png`: 老鼠图片（对应文本：三只老鼠）。
  - `assets/fill/jinYu.png`: 金鱼图片（对应文本：四只金鱼）。
  - `assets/fill/qingWa.png`: 青蛙图片（对应文本：三只青蛙）。
  - `assets/reset.png`: 重置按钮图标。
- **功能**:
  - 页面底部展示三个图片选项。
  - **点击或拖拽交互**: 用户既可以点击图片选项，也可以将图片**拖拽**到空格处，将其对应的描述文字填入句子中间，并伴有简单的强调动画。
  - 支持点击重置按钮清空填空内容。

## 6. 垃圾分类 (fenlei.html)

- **内容说明**: 一个垃圾分类拖拽游戏。
- **使用图片**:
  - `assets/fenlei/kehuishou.png`: 可回收垃圾桶。
  - `assets/fenlei/chuyu.png`: 厨余垃圾桶（同时也是分类项）。
  - `assets/fenlei/yilaguan.png`, `tshirt.png`: 可回收垃圾项。
  - `assets/fenlei/chuyu.png`, `gutou.png`: 厨余垃圾项。
  - `assets/fenlei/dianchi.png`: 其他垃圾项（干扰项）。
  - `assets/reset.png`: 重置按钮图标。
- **功能**:
  - **拖拽交互**: 页面展示可回收和厨余两个垃圾桶（位于顶部并行排列），底部展示多种垃圾物品。
  - **分类逻辑**:
    - 易拉罐、旧衣服 -> 拖入蓝色可回收桶。
    - 厨余、骨头 -> 拖入厨余垃圾桶。
    - 电池 -> 干扰项，无对应桶。
  - **判定反馈**:
    - 放入正确：物品消失并提示正确。
    - 放入错误：物品弹回并提示错误。
  - **胜利判定**: 收集完所有可回收和厨余物品后，显示“垃圾分类完成”的庆祝信息。
